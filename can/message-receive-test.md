# CAN-002 CAN 报文接收测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-002 |
| 用例名称 | CAN 报文接收测试 |
| 优先级 | P0 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. DUT 已上电 2. CAN 分析仪已连接 |

## 测试目的

验证 DUT 能够正确接收和处理 CAN 报文

## 测试步骤

1. 连接 CAN 分析仪到 DUT 的 CAN 接口
2. 配置 CAN 分析仪参数（波特率 500kbps）
3. 使用 CAN 分析仪发送测试报文
4. 验证 DUT 是否正确接收报文
5. 检查 DUT 内部寄存器/变量值
6. 测试不同 ID 报文的接收过滤

## 预期结果

- DUT 正确接收目标报文
- 接收数据与发送数据一致
- 报文过滤功能正常工作
- 接收中断/标志位正确触发
