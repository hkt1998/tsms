# CAN-007 CAN 总线边界测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-007 |
| 用例名称 | CAN 总线边界测试 |
| 优先级 | P2 |
| 测试类型 | 边界测试 |
| 预置条件 | 1. 可变参数 CAN 发生器已准备 |

## 测试目的

测试 CAN 总线在边界条件下的表现

## 测试步骤

1. 测试最低 baud 率（100kbps）
2. 测试最高 baud 率（1Mbps）
3. 测试最小数据长度（0 字节）
4. 测试最大数据长度（8 字节）
5. 测试最小 ID（0x000）
6. 测试最大 ID（0x7FF 标准帧）
7. 测试扩展帧最大 ID（0x1FFFFFFF）

## 预期结果

- 所有边界条件下功能正常
- 参数范围内无异常
