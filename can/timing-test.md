# CAN-006 CAN 总线时序测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-006 |
| 用例名称 | CAN 总线时序测试 |
| 优先级 | P1 |
| 测试类型 | 性能测试 |
| 预置条件 | 1. 示波器已准备 |

## 测试目的

测量和验证 CAN 总线时序参数

## 测试步骤

1. 使用示波器捕获 CANH 和 CANL 波形
2. 测量位时间
3. 测量传播延迟
4. 测量采样点位置
5. 测量跳变沿时间
6. 验证与理论值的一致性

## 预期结果

- 位时间误差 < 0.5%
- 采样点在 75%-80% 范围内
- 跳变沿符合规范要求
