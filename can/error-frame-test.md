# CAN-004 CAN 错误帧测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-004 |
| 用例名称 | CAN 错误帧测试 |
| 优先级 | P0 |
| 测试类型 | 可靠性测试 |
| 预置条件 | 1. CAN 分析仪支持错误注入 |

## 测试目的

验证 DUT 对 CAN 错误帧的处理能力

## 测试步骤

1. 注入 CRC 错误帧
2. 注入格式错误帧
3. 注入 ACK 错误帧
4. 注入位错误
5. 注入填充错误
6. 验证 DUT 错误计数器变化
7. 验证总线离线恢复功能

## 预期结果

- DUT 正确识别各类错误帧
- 错误计数器正确递增
- 达到错误阈值后进入总线离线
- 错误恢复后自动重连
