# CAN-001 CAN 报文发送测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-001 |
| 用例名称 | CAN 报文发送测试 |
| 优先级 | P0 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. DUT 已上电 2. CAN 分析仪已连接 |

## 测试目的

验证 DUT 能够正确发送 CAN 报文

## 测试步骤

1. 连接 CAN 分析仪到 DUT 的 CAN 接口
2. 配置 CAN 分析仪参数（波特率 500kbps）
3. 触发 DUT 发送 CAN 报文
4. 使用 CAN 分析仪捕获报文
5. 验证报文 ID、数据长度、数据内容
6. 测试标准帧和扩展帧发送

## 预期结果

- CAN 报文成功发送
- 报文 ID 与预期一致
- 数据内容正确无误
- 报文格式符合 CAN 协议
