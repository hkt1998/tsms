## 测试目的

验证 DUT 对 CAN FD 的支持和兼容性

## 测试步骤

1. 测试 CAN FD 经典帧格式
2. 测试 CAN FD 扩展帧格式
3. 测试可变数据长度（0-64 字节）
4. 测试高速数据相位（2Mbps/5Mbps）
5. 测试与经典 CAN 的互操作性
6. 验证 CRC 校验功能

## 预期结果

- 支持 CAN FD 协议
- 支持 64 字节数据载荷
- 向下兼容经典 CAN
- 高速相位工作正常
