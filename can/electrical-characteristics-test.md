# CAN-009 CAN 总线电气特性测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | CAN-009 |
| 用例名称 | CAN 总线电气特性测试 |
| 优先级 | P0 |
| 测试类型 | 电气测试 |
| 预置条件 | 1. 示波器和万用表已准备 |

## 测试目的

验证 CAN 总线电气特性符合规范

## 测试步骤

1. 测量 CANH 对地电压（显性）
2. 测量 CANL 对地电压（显性）
3. 测量 CANH 对地电压（隐性）
4. 测量 CANL 对地电压（隐性）
5. 测量差分电压
6. 测量终端电阻（应为 60 欧姆）
7. 测量共模电压范围

## 预期结果

- 显性 CANH：约 3.5V
- 显性 CANL：约 1.5V
- 隐性 CANH/CANL：约 2.5V
- 差分电压：显性>1.5V，隐性<0.5V
- 终端电阻：60±3 欧姆

## 备注

符合 ISO 11898 标准
