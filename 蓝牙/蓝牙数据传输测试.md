| 项目 | 内容 |

| 用例编号 | BT-003 |

| 优先级 | P0 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. DUT 与测试设备已建立蓝牙连接 |

## 测试目的

验证蓝牙数据传输功能的正确性和完整性

## 测试步骤

1. 建立蓝牙连接
2. 从测试设备向 DUT 发送测试文件（1MB）
3. 验证 DUT 接收文件的完整性
4. 从 DUT 向测试设备发送测试文件（1MB）
5. 验证测试设备接收文件的完整性
6. 重复步骤 2-5，测试不同大小的文件（10KB, 100KB, 10MB）

## 预期结果

- 所有文件传输成功
- 接收文件与发送文件完全一致（MD5 校验通过）
- 传输速率符合蓝牙规格要求
