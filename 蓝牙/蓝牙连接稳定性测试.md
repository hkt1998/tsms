# BT-002 蓝牙连接稳定性测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | BT-002 |
| 用例名称 | 蓝牙连接稳定性测试 |
| 优先级 | P0 |
| 测试类型 | 稳定性测试 |
| 预置条件 | 1. DUT 与测试设备已完成配对 |

## 测试目的

验证蓝牙连接在长时间运行下的稳定性

## 测试步骤

1. 建立 DUT 与测试设备的蓝牙连接
2. 保持连接状态持续运行 24 小时
3. 每隔 1 小时记录连接状态
4. 期间进行间歇性数据传输
5. 记录任何断连事件

## 预期结果

- 24 小时内无异常断连
- 数据传输成功率 100%
- 连接延迟保持在正常范围内
