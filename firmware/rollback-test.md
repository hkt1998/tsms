# FW-003 固件回滚测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | FW-003 |
| 用例名称 | 固件回滚测试 |
| 优先级 | P0 |
| 测试类型 | 可靠性测试 |
| 预置条件 | 1. 旧版本固件已准备 |

## 测试目的

验证固件升级失败后的回滚功能

## 测试步骤

1. 升级到新版本
2. 模拟升级失败（如校验失败）
3. 验证自动回滚到旧版本
4. 手动触发回滚
5. 验证回滚后版本号
6. 验证回滚后功能正常

## 预期结果

- 升级失败自动回滚
- 手动回滚成功
- 回滚后系统正常
- 数据完整性保持
