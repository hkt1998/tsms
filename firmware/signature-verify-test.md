# FW-005 固件签名验证测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | FW-005 |
| 用例名称 | 固件签名验证测试 |
| 优先级 | P0 |
| 测试类型 | 安全测试 |
| 预置条件 | 1. 签名密钥对已生成 |

## 测试目的

验证固件数字签名机制

## 测试步骤

1. 使用私钥签名固件
2. 验证签名通过
3. 使用错误密钥签名
4. 验证签名失败
5. 测试签名被篡改的情况
6. 验证无签名固件被拒绝

## 预期结果

- 有效签名验证通过
- 无效签名验证失败
- 签名被拒绝启动
- 符合安全启动要求

## 备注

使用 RSA 或 ECDSA 签名算法
