# BT-007 蓝牙断连重连测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | BT-007 |
| 用例名称 | 蓝牙断连重连测试 |
| 优先级 | P0 |
| 测试类型 | 可靠性测试 |
| 预置条件 | 1. DUT 与测试设备已配对 |

## 测试目的

验证蓝牙断连后的自动重连功能

## 测试步骤

1. 建立 DUT 与测试设备的蓝牙连接
2. 关闭测试设备蓝牙，模拟断连
3. 等待 10 秒
4. 重新打开测试设备蓝牙
5. 观察 DUT 是否自动重连
6. 重复步骤 2-5 共 10 次
7. 记录每次重连成功率和时间

## 预期结果

- 每次重连成功率 100%
- 平均重连时间 < 5 秒
- 无需手动干预即可完成重连
