# BT-004 蓝牙功耗测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | BT-004 |
| 用例名称 | 蓝牙功耗测试 |
| 优先级 | P1 |
| 测试类型 | 性能测试 |
| 预置条件 | 1. 功耗测试设备已连接 2. DUT 电池充满 |

## 测试目的

测量蓝牙功能在不同工作状态下的功耗

## 测试步骤

1. DUT 待机状态，测量基础功耗
2. 打开蓝牙（未连接），测量扫描功耗
3. 建立蓝牙连接，测量连接功耗
4. 进行数据传输，测量传输功耗
5. 进入蓝牙低功耗模式，测量睡眠功耗
6. 记录各状态下 1 分钟内的平均电流

## 预期结果

- 待机功耗：< 1mA
- 扫描功耗：< 10mA
- 连接功耗：< 15mA
- 传输功耗：< 30mA
- 睡眠功耗：< 0.1mA
