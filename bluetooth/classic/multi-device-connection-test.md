# BT-006 蓝牙多设备连接测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | BT-006 |
| 用例名称 | 蓝牙多设备连接测试 |
| 优先级 | P1 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. 准备 3 台以上蓝牙测试设备 |

## 测试目的

验证 DUT 同时连接多个蓝牙设备的能力

## 测试步骤

1. DUT 与设备 A 配对并连接
2. DUT 与设备 B 配对并连接
3. DUT 与设备 C 配对并连接
4. 验证所有设备同时保持连接状态
5. 同时向所有设备发送数据
6. 断开设备 B，验证 A 和 C 保持连接
7. 重新连接设备 B

## 预期结果

- 所有设备可同时保持连接
- 多设备数据传输互不干扰
- 单设备断连不影响其他设备

## 备注

最大连接数根据具体蓝牙规格而定
