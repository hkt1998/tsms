# ETH-001 以太网链路层测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | ETH-001 |
| 用例名称 | 以太网链路层测试 |
| 优先级 | P0 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. DUT 已上电 2. 以太网分析仪已连接 |

## 测试目的

验证以太网链路层基本功能

## 测试步骤

1. 连接以太网分析仪到 DUT
2. 测试链路自动协商（100M/1000M）
3. 测试全双工/半双工模式
4. 发送测试帧验证 MAC 地址
5. 测试 VLAN 标签功能
6. 验证帧校验序列（FCS）

## 预期结果

- 链路协商成功
- MAC 地址正确
- VLAN 功能正常
- FCS 校验通过

## 备注

符合 IEEE 802.3 标准
