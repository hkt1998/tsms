# ETH-004 TSN 时间敏感网络测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | ETH-004 |
| 用例名称 | TSN 时间敏感网络测试 |
| 优先级 | P1 |
| 测试类型 | 性能测试 |
| 预置条件 | 1. TSN 测试环境已搭建 |

## 测试目的

验证时间敏感网络功能

## 测试步骤

1. 测试时钟同步（802.1AS）
2. 测试流量调度（802.1Qbv）
3. 测试帧抢占（802.1Qbu）
4. 测试流预留协议（802.1Qat）
5. 测量端到端延迟
6. 验证延迟抖动

## 预期结果

- 时钟同步精度<1us
- 低优先级流量可被抢占
- 端到端延迟可预测
- 抖动在允许范围内

## 备注

符合 IEEE 802.1 TSN 标准
