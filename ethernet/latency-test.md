## 测试目的

测量以太网通信延迟

## 测试步骤

1. 发送小包（64 字节），测量往返延迟
2. 发送中包（512 字节），测量往返延迟
3. 发送大包（1500 字节），测量往返延迟
4. 在高负载下重复测试
5. 统计平均延迟、最小延迟、最大延迟
6. 计算延迟抖动

## 预期结果

- 小包延迟 < 100us
- 大包延迟 < 500us
- 延迟抖动 < 50us
