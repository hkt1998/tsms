# ETH-010 以太网压力测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | ETH-010 |
| 用例名称 | 以太网压力测试 |
| 优先级 | P1 |
| 测试类型 | 压力测试 |
| 预置条件 | 1. 压力测试环境已搭建 |

## 测试目的

验证以太网在长期高负载下的稳定性

## 测试步骤

1. 设置 95% 带宽利用率
2. 持续发送混合类型流量
3. 持续运行 48 小时
4. 每小时记录性能指标
5. 监控丢包率和延迟
6. 测试结束后验证功能

## 预期结果

- 48 小时无系统崩溃
- 丢包率 < 0.01%
- 延迟稳定
- 测试后功能正常
