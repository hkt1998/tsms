# ETH-007 AVB 音频视频桥接测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | ETH-007 |
| 用例名称 | AVB 音频视频桥接测试 |
| 优先级 | P2 |
| 测试类型 | 功能测试 |
| 预置条件 | 1. AVB 测试设备已准备 |

## 测试目的

验证音频视频流传输功能

## 测试步骤

1. 测试 AVB 流预留
2. 测试音频流传输
3. 测试视频流传输
4. 测量音视频同步
5. 测试流带宽保证
6. 验证丢包率

## 预期结果

- 音视频流畅传输
- 同步误差 < 20ms
- 丢包率 < 0.001%
- 带宽得到保证

## 备注

符合 IEEE 802.1 AVB 标准
