# ETH-005 以太网带宽测试

## 测试用例信息

| 项目 | 内容 |
|-----|------|
| 用例编号 | ETH-005 |
| 用例名称 | 以太网带宽测试 |
| 优先级 | P1 |
| 测试类型 | 性能测试 |
| 预置条件 | 1. 带宽测试工具已准备 |

## 测试目的

测量以太网实际可用带宽

## 测试步骤

1. 建立 TCP 连接
2. 使用 iperf 进行带宽测试
3. 测试 100Mbps 模式带宽
4. 测试 1000Mbps 模式带宽
5. 测试 UDP 吞吐量
6. 记录不同包大小下的吞吐量

## 预期结果

- 100M 模式：>90Mbps
- 1000M 模式：>900Mbps
- UDP 吞吐量接近理论值
